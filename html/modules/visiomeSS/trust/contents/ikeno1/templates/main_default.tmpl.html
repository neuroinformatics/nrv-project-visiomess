Current clamp simulation of Hodgkin-Huxley Model

Hidetoshi IKENO
ikeno@hept.himeji-tech.ac.jp

Reference Paper

A. L. Hodgkin and A. F. Huxley (1952), A quantitative description of membrane current and its application to conduction and excitation in nerve, J. Physiol. (Lond.), Vol.117, 500-544

This package contains the following files:

README.txt
This file.
hh.mdl
Model descriptions written in NCS language.
sim.sl
SATELLITE shell script file for running the model.

HOW TO RUN

  1. This package is running with SATELLITE system developed at Biological & Physiological Engineering Laboratory, Toyohashi University of Technology. SATELLITE will be found in the followings:
    • http://satellite.sourceforge.jp/
  2. Put "hh.mdl" and "sim.sl" in the same working directory.
  3. In the working directory, run SATELLITE by "sl" command.
  4. Execute the script by "inline("sim.sl")".
