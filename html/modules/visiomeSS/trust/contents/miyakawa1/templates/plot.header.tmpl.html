Dependence of linearity on the inter-stimulus interval under blockade of GABA receptors

You can plot dependence of the linearity on the ISI in the various conditions. Click "Plot Data", then "View Result" for plotting graph. "Edit Plot Parameters" button is for edit scales for X and Y axis. "Reset" button is for reset parameters to default values.

  • xmin: minimum value of X axis
  • xmax: maximum value of X axis
  • ymin: minimum value of Y axis
  • ymax: minimum value of Y axis
