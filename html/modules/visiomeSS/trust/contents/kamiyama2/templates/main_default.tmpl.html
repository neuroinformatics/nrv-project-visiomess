Revised Hodgkin-Huxley Model by Clay

Yoshimi Kamiyama
kamiyama@ist.aichi-pu.ac.jp

Original Report

Clay, J, R. (1998), Excitability of the Squid Gian Axon Revisited, Journal of Neurophysiology, Vol.80, pp.903-913 PubMed: 9705477

This package contains the following files:

README.txt
This file.
hh.mdl
Classical HH Model written in NCS language.
hh.sl
SATELLITE shell script file for the Classical HH.
clay.mdl
Revised HH Model written in NCS language.
cray.sl
SATELLITE shell script file for the Revised HH.
hh-cray.sl
SATELLITE shell script file for running the models.

HOW TO RUN

  1. This package is running with SATELLITE system developed at Biological & Physiological Engineering Laboratory, Toyohashi University of Technology. SATELLITE will be found in the followings:
    • http://satellite.sourceforge.jp/
  2. Put all files in the same working directory.
  3. In the working directory, run SATELLITE by "sl" command.
  4. Execute the script by "inline("hh-clay.sl")".

The simulation results as in Figure 3 of the reference paper will be displayed on the screen.
