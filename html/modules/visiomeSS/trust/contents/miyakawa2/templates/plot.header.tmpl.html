Dendritic attenuation of synaptic potenstiation in hippocampal CA1 pyramidal neurons

You can plot EPSP in hippocampal CA1 pyramidal neurons. Click "Plot Data", then "View Result" for plotting graph. "Edit Plot Parameters" button is for edit scales for X and Y axis. "Reset" button is for reset parameters to default values.

  • xmin: minimum value of X axis
  • xmax: maximum value of X axis
  • ymin: minimum value of Y axis
  • ymax: minimum value of Y axis
