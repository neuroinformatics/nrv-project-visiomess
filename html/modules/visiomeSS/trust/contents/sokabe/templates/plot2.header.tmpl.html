Single channel current trace

You can plot single channel current traces under symmetric 500 mM KCl at different voltages. Click "Plot Data", then "View Result" for plotting graph. "Edit Plot Parameters" button is for edit scales for time and current amplitude axis. "Reset" button is for reset parameters to default values.

  • xmin: minimum value of time axis
  • xmax: maximum value of time axis
  • ymin: minimum value of current amplitude axis
  • ymax: minimum value of current amplitude axis
